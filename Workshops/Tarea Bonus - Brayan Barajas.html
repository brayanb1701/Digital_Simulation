Para simular la cinmatica de una partcula en 2D se necesita tener en cuenta tanto la componente en x como en y, por lo que se puede tener posicin, velocidad y aceleracin para cada una de estas.

En el ejercicio planteado, se observa que no hay componente de aceleracin en x. Teniendo en cuenta esto, se podra describir este sistema por medio de las siguientes ecuaciones de diferencia:

  • $s_x(t+dt)= s_x(t) + dt*v_x(t) + otros$
  • $s_y(t+dt)= s_y(t) + dt*v_y(t) + otros$
  • $v_y(t+dt) = v_y(t) + dt*a_y(t) + otros$

#Variables que se pueden ajustar al sistema
x_0 = 0 #@param {type:"number"}
y_0 = 10 #@param {type:"number"}
vx_0 = -2 #@param {type:"number"}
vy_0 =  26#@param {type:"number"}
left_limit = -10 #@param {type:"number"}
right_limit = 10 #@param {type:"number"}
top_limit = 40 #@param {type:"number"}
bottom_limit = 0 #@param {type:"number"}
delta_t = delta

#Vectores donde se almacenarn todos los valores de velocidad y posicin
all_vx = np.zeros(n_steps+1)
all_vy = np.zeros(n_steps+1)
all_x = np.zeros(n_steps+1)
all_y = np.zeros(n_steps+1)           

#Inicializacin de variables
x,y,vx,vy=x_0,y_0,vx_0,vy_0

for i in range(n_steps):
    #Ecuaciones de diferencia
    x=x + vx*delta_t
    y=y + vy*delta_t
    vy = vy + ay[i]*delta_t
    #Condicin de rebote en lmites verticales
    if x<=left_limit or x>=right_limit: 
        vx=-vx*0.7 #Disminucin de velocidad en x
        x = left_limit if x<=left_limit else right_limit #Restriccin para no pasar de los lmites
    #Condicin de rebote en lmites horizontales 
    if y<=bottom_limit or y>=top_limit:
        vy=-vy*0.7 #Disminucin de velocidad en x
        y=bottom_limit if y<=bottom_limit else top_limit #Restriccin para no pasar de los lmites

    #Guardar los valores en cad momento de tiempo
    all_vx[i] = vx
    all_vy[i] = vy
    all_x[i] = x
    all_y[i] = y
